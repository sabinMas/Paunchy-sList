## Quick CDN URL Patterns

### For Simple Icons
Replace `[SLUG]` with icon slug:
```
https://cdn.jsdelivr.net/npm/simple-icons@latest/icons/[SLUG].svg
```

Examples:
- `github` → `https://cdn.jsdelivr.net/npm/simple-icons@latest/icons/github.svg`
- `docker` → `https://cdn.jsdelivr.net/npm/simple-icons@latest/icons/docker.svg`
- `python` → `https://cdn.jsdelivr.net/npm/simple-icons@latest/icons/python.svg`

### For Tabler Icons
Replace `[NAME]` with icon name:
```
https://cdn.jsdelivr.net/npm/@tabler/icons@latest/icons/[NAME].svg
```

Examples:
- `tag` → `https://cdn.jsdelivr.net/npm/@tabler/icons@latest/icons/tag.svg`
- `database` → `https://cdn.jsdelivr.net/npm/@tabler/icons@latest/icons/database.svg`
- `settings` → `https://cdn.jsdelivr.net/npm/@tabler/icons@latest/icons/settings.svg`

## How to Use

1. **Direct image embedding:**
   ```html
   <img src="https://cdn.jsdelivr.net/npm/simple-icons@latest/icons/github.svg" 
        alt="GitHub" width="24">
   ```

2. **In JavaScript:**
   ```javascript
   const iconUrl = 'https://cdn.jsdelivr.net/npm/simple-icons@latest/icons/github.svg'
   const img = document.createElement('img')
   img.src = iconUrl
   img.alt = 'GitHub'
   ```

3. **With fallback:**
   ```javascript
   img.onerror = () => {
     img.textContent = '🤖' // fallback emoji
   }
   ```

4. **CSS background:**
   ```css
   .github-icon {
     background: url('https://cdn.jsdelivr.net/npm/simple-icons@latest/icons/github.svg');
     width: 24px;
     height: 24px;
   }
   ```
